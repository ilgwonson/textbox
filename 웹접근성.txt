1. 이미지 대체 텍스트
    - alt 속성을 통해 제공
    - 내용이 너무 길경우 longdesc 속성 사용 ex) longdesc="graph_desc.html"
    - QR코드는 alt에 내용및 url을 적거나 QR코드 기능 작동하는 버튼을 따로 구현한다.
    - 내용과 상관없는 이미지 일 경우 alt내용 빈공간으로 만든다. ex) alt=""
2. w3c 유효성 검사
    - 돔구조가 잘못되면 스크린 리더가 재대로 읽지 못함
3. 데이터 테이블
    - caption으로 테이블 제목 명시
    - summary로 테이블 내용 요약
    - scope로 행과 열 연결구조 명시
4. 논리적 순서
    - html dom 구조를 논리적 순서에 맞게 구현한다.
5. 주 언어 명시
    - lang="ko" html에 표시하고
    - 언어가 바뀌때마다 재 선언 해준다.
6. 키보드 논리적 순서
    - 키보드로 레이어가 떳을 경우 레이어로 포커스 이동
    - 플래시는 제작할때 커서이동에 대한 값 설정
    - 에디터의 경우 포거스를 바꾸는 단축키 생성
7. 키보드 포커스
    - 탭으로 이동시 순서적으로 이동 가능
    - outilne 속성 없애면 안됨
8. 프레임 타이틀
    - 프레임에 타이틀 넣어주기
9. 콘텐츠 블록 제목
    - 헤더태그를 통해 문서를 구조적으로 만듬
10. 링크 텍스트
    - 더보기 같은 경우 어떤 더보기인지 blind css 등을 통한 표시
    - bilnd css 는 display:none이 아닌 안보이는 요소로 만들어 주는걸로 바꿈
11. 폼요소와 레이블
    - label과 input을 묵어준다.
12. 사용자 입력 오류 알림
    - form validation에서
    - 잘못된 내용 경고창으로 상세하게 알려준다.
    - 잘못된 곳으로 포커스를 자동으로 이동시켜 준다.
